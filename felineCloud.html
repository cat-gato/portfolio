Feline Cloud

A website that has been in the works for way too long. This project will provide hosting services for both game, and web servers. When I was younger and wanted to host my own website, it interested me how these services worked and functioned in the back-end. I eventually learned how they worked and setup a small private hosting infrastructure that I have used to host personal projects and services, but also hosting for close friends and family. Working on this over the years was a great source of learning about a lot of different things, from using Linux daily, to learning programming.
